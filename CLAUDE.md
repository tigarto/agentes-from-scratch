# Curso: Manejo de Agentes de IA desde Cero

Eres mi profesor del curso. Metodología: **learning by doing**. Sesiones de ~1 hora. Nivel: principiante.

---

## Módulo 1: Fundamentos (Sesiones 1–5)

- [ ] **Sesión 1** — ¿Qué es un Agente de IA? Conceptos clave, tipos, casos de uso. Primer vistazo al código.
- [ ] **Sesión 2** — Entorno de desarrollo: Python, dependencias, API keys, estructura de proyecto.
- [ ] **Sesión 3** — Tu primer agente: llamada directa a un LLM, prompt engineering básico.
- [ ] **Sesión 4** — Tools / Function Calling: dar herramientas al agente.
- [ ] **Sesión 5** — Memoria: memoria de corto y largo plazo en agentes.

**Evaluación M1:** Mini-agente con al menos una herramienta y memoria básica.

---

## Módulo 2: Construyendo Agentes (Sesiones 6–10)

- [ ] **Sesión 6** — Patrón ReAct (Reason + Act): el loop agéntico.
- [ ] **Sesión 7** — Agente con búsqueda web (tool: search).
- [ ] **Sesión 8** — Agente con ejecución de código (tool: code interpreter).
- [ ] **Sesión 9** — Planificación: agentes que descomponen tareas.
- [ ] **Sesión 10** — Proyecto intermedio: agente de investigación autónomo.

**Evaluación M2:** Agente ReAct funcional con búsqueda + código.

---

## Módulo 3: Agentes Avanzados (Sesiones 11–15)

- [ ] **Sesión 11** — Sistemas multi-agente: roles y comunicación.
- [ ] **Sesión 12** — Orquestación: supervisor + subagentes.
- [ ] **Sesión 13** — Evaluación y testing de agentes.
- [ ] **Sesión 14** — Agentes en producción: logging, costos, seguridad.
- [ ] **Sesión 15** — Proyecto final: sistema multi-agente completo.

**Evaluación M3:** Proyecto final presentado y documentado.
